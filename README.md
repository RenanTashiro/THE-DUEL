# THE-DUEL
A two player console game

## Dependences
ncurse library

```
sudo apt-get install libncurses-dev
```

## How to play
Just compile de duel.cpp file and execute

```
g++ -std=c++11 -o duel duel.cpp -lncurses
./duel
```
